Semantic Tags

  • <nav>: Used for websites navigation
  • <main>: Used for the main content of a page
  • <section>: Defines a section in a document. A section is a thematic grouping of content. Examples:
    • Introduction
    • Content
    • Contact information
  • <article>: Specific indenpendent and self-contained content
  • <footer>: Specififed a footer for a document or section
  • <aside>: Defines contnet aside from the content it is placed in (e.g. sidebar)
  • For more semantic elements, go here

There are several content tags that are used for formatting text. These tags are not block tags. This means that they are used inline to format text inside a block.
We can make text bold, strong, italic or emphasied. Other ways of emphasizing text in include highlighting it or underlining it. We can also change text to a smaller size, write it as a subscript , a superscript or crossed out .
